Seashell User Manual
Welcome to Seashell, a simple UNIX shell with basic functionality. This manual provides an overview of how to use the shell, including its commands and features.

Getting Started
To use Seashell, open a terminal and navigate to the directory containing the makefile executable file.
Run "make" in the terminal once to compile all necessary file to run the shell.
Then, type the command ./myshell to start the shell.

Commands
myshell supports the following internal commands:

cd
Use cd to change the current default directory. If no argument is provided, an error message will be shown. If the directory provided does not exist, an appropriate error message will be displayed. The cd command also changes the PWD environment variable.
cd <directory>

clr
Use clr to clear the screen.

dir
Use dir to list the contents of a directory. This command is equivalent to ls -al.
dir <directory>

environ
Use environ to display all the environment strings.

echo
Use echo to display a message on the screen. Multiple spaces or tabs may be reduced to a single space.
echo <message>

help
Use help to display the user manual using the more filter.

pause
Use pause to pause the operation of the shell until the Enter key is pressed.

quit
Use quit to exit the shell.
